# Quasar App (Quasar Pomoro Timer)

## Demo

See the [live demo](http://quasarpomodorotimer.surge.sh/#/).

## Install the dependencies
```bash
npm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```
